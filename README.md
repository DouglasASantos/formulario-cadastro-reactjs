## Como executar a aplicação 


- Caso não tenha instalado no seu terminal o pacote npm, execute o seguinte comando:

```

 npm install

```

- Navegar até a pasta executando:

```

 cd ./project

```

- Executar o projeto através do comando npm start:

```

npm start

```
